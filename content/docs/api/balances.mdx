## Get account balances

**GET** <code><ApiUrl env="production" />/account/balances</code>

Retrieves account balance information. By default this endpoint returns balances
for **all currencies**.

### Single-currency lookup

Append a currency code to the path to fetch a single balance:

```
GET /account/balances/inr
```

### Path parameters

| Parameter  | Type   | Notes                                                                |
| ---------- | ------ | -------------------------------------------------------------------- |
| `currency` | string | Optional. Currency code (for example, `inr`, `usdt`).                |

### Responses

| Code  | Meaning                                       |
| ----- | --------------------------------------------- |
| `200` | Balance information retrieved successfully.   |
| `404` | Currency not found (single-currency lookup).  |

#### Response fields — `BalanceResponse`

| Field         | Type   | Description                                                              |
| ------------- | ------ | ------------------------------------------------------------------------ |
| `currency`    | string | Currency code (for example, `inr`, `usdt`).                              |
| `balance`     | string | Total balance held for this currency (string-encoded number).            |
| `locked`      | string | Portion of the balance committed to in-flight payouts.                   |
| `escrow`      | string | Portion held in escrow against pending settlement.                       |
| `wallet_type` | string | `fiat` or `crypto`.                                                      |
